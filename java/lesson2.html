Java Lesson 1

Overview of Computer Programming

What is Computer Programming?

If you are completely new to computer programming, there is no better starting point than right here. You might have already scrolled ahead to see what quick lines of code you can pick up, but in this lesson, we would rather like to direct your attention to what computer programming really is.

Computer programming is a process, starting from the design of a solution to a particular task to its eventual build, which when done right, will result in success. Another way to put it is that computer programming is a way of problem solving. You will often be tasked with something that the computer must do, and it's your job to figure out how to communicate with the computer on how to accomplish that task as efficiently as possible.

Communicate? Yes, that's right. While you might not be literally talking to a screen (although nothing wrong with that), coding is essentially a written form of communicating with your computer. Imagine writing a recipe guide for a friend which provides all the instructions needed to create a certain dish. Programming has the same underlying idea: you give the computer “instructions” (lines of code) in order to create a certain dish (usually a computer program to accomplish a certain task).

Why Learn Java?

Just like how English is one of many languages in the world, Java is one of many coding languages out there. You might be wondering why Java in particular is worth the space in your brain. Well, here are a few reasons why:

  • It’s a very common language used in the programming world and knowing how to code in Java is a useful skill to be able to whip out
  • It gives its developers access to a vast API (this will be discussed later in the course)
  • It is a language centered around Object Orientated Programming (no need to know what this means now, this will also be elaborated on later) which brings very powerful uses to programs
  • It runs faster than many other languages such as Python
  • It’s a good language to know in order to learn more complex languages needed for app development
  • It’s free! Free is always nice

Now that you hopefully know what and why you might be here to learn how to code in Java, the next step is to download all the necessary components to get you set up for coding your first Java program!
